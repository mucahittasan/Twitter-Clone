<h2> ğŸ” Setup</h2>

- Clone the project with **"git clone"**

- After cloning the project, by following these steps, you will fulfill the project requirements.

  ```npm
  npm i
  ```

  - install with yarn:

  ```yarn
  yarn add
  ```
